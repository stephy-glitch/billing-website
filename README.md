# ChaatGPT Restaurant Billing System

A simple, offline billing system for ChaatGPT restaurant. This system allows you to create bills, manage orders, and print receipts without requiring an internet connection.

## Features

- **Complete Menu**: All items from both ChaatGPT menus included
- **Easy Order Management**: Add items, adjust quantities, and remove items
- **Packing Charge**: Optional packing charge (₹5 per item)
- **Bill Generation**: Print professional bills with itemized details
- **Offline**: Works completely offline - no internet required
- **Responsive Design**: Works on desktop and tablet devices

## How to Use

1. **Open the System**: 
   - Simply open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.)
   - No installation or server required

2. **Add Items to Order**:
   - Click the "Add" button next to any menu item
   - Items will appear in the Order Summary section on the right

3. **Manage Order**:
   - Use the `+` and `-` buttons to adjust quantities
   - Click "Remove" to delete an item from the order
   - Check the "Apply Packing Charge" checkbox if needed (₹5 per item)

4. **View Totals**:
   - Subtotal: Sum of all items
   - Packing Charge: ₹5 × number of items (if enabled)
   - Total: Final amount to be paid

5. **Print Bill**:
   - Click "Print Bill" button to generate and print a professional receipt
   - The bill includes:
     - Restaurant name and contact information
     - Date and time
     - Unique bill number
     - Itemized list with quantities and prices
     - Subtotal, packing charge, and total

6. **Clear Order**:
   - Click "Clear Order" to start a new order
   - Confirmation prompt will appear to prevent accidental clearing
